# Daryl Simene Portfolio

Personal portfolio website built with React, TypeScript, Vite, Tailwind CSS, and React Router.

## Features

- Responsive landing page
- Portfolio preview and full portfolio page
- Project modal with image carousel
- Smooth navigation
- Contact section and contact modal
- Custom dark UI with glow effects

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React
- React Icons

## Getting Started

```bash
npm install
npm run dev
```
